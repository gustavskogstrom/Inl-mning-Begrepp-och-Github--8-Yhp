Github flow

Vad är det? Hur använder man det? Vilka viktiga delar är inkluderade i begreppet?

Github flow är något som man använder för att skapa branches för att sedan kunna arbeta smidigt med andra.

Git, github flödet Flödet i git och github är kallad för github flow eller gitflow, hur vi använder flödet i github. Man använder det till kod hantering där man delar med sig av sin kod med andra(collaberators) det man ändrar kallas för flödet. Hur gör man? Jo man startar med att skapa en strucktur i github som man sedan laddar ner. När de är klara så puchar de upp sina repon så att alla har användning för det, till sist så mergar man dem. Att merga betyder att man lägger ihop dem till ett repo.

Flexbox

Vad är det? Hur fungerar det?

Flexbox är en stark funktion som kan manipulera ett HTML-elements positioner. En flexbox kan manipulera storlekar och anpassa gränssnitt. Normalt om man sätter dit en flexbox så plaseras den vågrät (row) men om man vill ändra riktning eller har boxes i en displaybox så kan du använda column istället.

DOM (Document Object Model)

Vad är det? Hur manipulerar man DOM?

DOM står för Document Object Mode. DOM är en modell som repressenterar alla element som object. Objecten(elementen) kan manipuleras. Vi kan manipulera objecten genom att kalla på dem i Script.js genom att skriva const variabel document.getElementById("element"): Sedan ändrar du i din HTML genom att skriva variabel.innerHTML ="Det som du skriver här syns i din HTML."; Det är så du manipulerar DOM.

Fin kod

Vad menas med det begreppet? Varför är det viktigt?

Fin kod annars vid namn snygg kod betyder att koden är lätt läst och strukturerad. Det är viktigt att ha Fin kod för att andra som ska läsa din kod ska förstå den samt att det går fortare/effektivare.

Ramverk

Inom webbutveckling använder man begreppet ramverk, vad är det? Beskriv vad det är och berätta lite mer om två valfria.

Ramverk är ett packet med färdig kod som man plockar kod från när man kodar för att det ska gå effektivare att koda, alltså det är kod som någon annan redan har skrivit. Bootstrap och react.js är två ramverk. Bootstrap är ett populärt ramverk som används för att skapa webbsidor, det är färdig kod som man plockar eller laddar ner från nätet. I den koden så finns det färdiga delar av webbsidor som är lätt att använda. React.js är ett Javascript-bibliotek med färdig kod, detta ramverk användes för att skapa Facebook och Instagram.
